# Drift Net Improved

**Important**: This plugin replaces the core Drift Net plugin. Make sure to disable the default "Drift Net" plugin when using this plugin.

An improved version of the RuneLite Drift Net plugin with additional customization options.

## New Features (compared to core plugin)

This plugin adds the following options (in addition to the core Drift Net plugin options)

- **Net Fill Opacity**: Adjustable fill opacity for drift nets (0-255) using the existing status color
- **Use Clickbox for Nets**: Option to highlight nets using clickbox instead of hull
- **Untagged Fish Fill Color**: Fill color for untagged fish tiles (in addition to the tile border)
- **Tagged Fish Highlighting**: Highlight tagged fish with customizable border and fill colors
- **Annette Fill Opacity**: Adjustable fill opacity for Annette highlighting (0-255)
- **Fish Timer Display**: Show timers on tagged fish (ticks, seconds, or pie timer)
- **Out of Nets Notification**: Get notified when you run out of drift nets
- **Full Net Notification**: Get notified when a drift net becomes full
- **Menu Entry Hiding**: Hide the 'Take down' option from empty drift nets
- **Menu Swap for Fish**: Prioritize untagged fish over tagged fish when they overlap, allowing you to left-click twice to tag both
- **Statistics Overlay**: Track total fish caught, session fish count, and fish per hour
- **Tacklebox Highlighting**: Highlight the tacklebox in your inventory when you have no drift nets
